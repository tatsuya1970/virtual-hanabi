# バーチャル花火大会 ホームページ 要件定義書

## 1. プロジェクト概要

### 1.1 目的
バーチャル花火大会の公式ホームページを制作し、以下の2つのビジネス目標を達成する。

1. **受注獲得**: 企業・自治体からバーチャル花火大会開催の受注を獲得する
2. **スポンサー獲得**: 協賛（オリジナル花火・テロップ）の申し込みを獲得する

### 1.2 ターゲットユーザー
| ターゲット | ニーズ | 導線 |
|-----------|--------|------|
| 自治体職員（花火大会担当） | 中止となった花火大会の代替案を探している | 開催プラン → お問い合わせ |
| 企業のイベント担当者 | 地域活性化・プロモーション手段を探している | 開催プラン → お問い合わせ |
| 一般ユーザー（花火大会参加者） | 開催予定の大会に参加したい | 開催予定 → clusterリンク |
| 協賛希望者（個人・企業） | 花火大会を応援したい・PRしたい | 協賛プラン → お問い合わせ |

### 1.3 主催者情報
- 主催者名: 武村
- メールアドレス: takemura@takemura-techlab.com

## 3. デザイン方針

### 3.1 全体コンセプト
- **ダーク系**: 夜空をベースにした暗い背景
- **アクセントカラー**: 花火を連想させるオレンジ・金色
- **印象**: 華やか・祝祭感・地域の温かみ

### 3.2 カラーパレット（案）
| 用途 | カラー | 備考 |
|------|--------|------|
| 背景（メイン） | #0a0a1a〜#1a1a2e | 夜空のグラデーション |
| 背景（セクション交互） | #111128〜#0d0d20 | セクション区切り |
| アクセント（プライマリ） | #f59e0b〜#f97316 | オレンジ・金色（花火） |
| アクセント（セカンダリ） | #ef4444〜#ec4899 | 赤・ピンク（花火の彩り） |
| テキスト（メイン） | #ffffff | 白 |
| テキスト（サブ） | #9ca3af〜#d1d5db | グレー |

### 3.3 フォント
- 見出し: Noto Sans JP（Bold / Black）
- 本文: Noto Sans JP（Regular）
- 英字アクセント: システムフォントまたは Google Fonts

### 3.4 レスポンシブ対応
- モバイルファースト設計
- ブレークポイント: sm(640px) / md(768px) / lg(1024px) / xl(1280px)

| 広島 | 路面電車に乗れる |

- PC（ブラウザ）
- スマートフォン（iOS / Android）
- VRヘッドセット
- プラットフォーム: cluster（クラスター）

### 4.6 セクション5: 開催プラン
企業・自治体向けの開催提案セクション。

#### 訴求ポイント
- 花火大会中止の代替案として地域イベントを継続できる
- リアル開催より低コスト（10万円〜）
- AR連携による現地ハイブリッド体験 + 自宅からのオンライン視聴
- 地域活性化・観光プロモーション
- 5年間・13回の開催実績

#### 開催フロー（ステップ表示）
1. お問い合わせ・ヒアリング
2. 企画・提案
3. ワールド制作（メタバース空間の構築）
4. 告知・集客サポート
5. イベント当日運営
6. 開催レポート・振り返り

#### 料金
- 開催費用: 10万円〜（規模・内容により変動）
- 「まずはお気軽にご相談ください」の文言
- CTAボタン: 「お問い合わせ」
